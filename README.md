# Анализ продаж медицинских пробирок
### Описание проекта: 
Цель проекта: создание дашборда для медицинской компании для анализа продаж медицинских пробирок
Используемые технологии: PostgreSQL, Yandex Datalens
Цель дашборда: Отслеживание метрик отдела продаж, оценка эффективности работы с поставщиками, каналами сбыта и покупателями.

Заказчик: Директор отдела продаж.

Целевая аудитория: Директор отдела продаж, менеджеры по продажам, менеджеры по работе с поставщиками.

Ссылка на дашборд: https://datalens.ru/qmrrfw7d1y6ab
