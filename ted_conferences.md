# Анализ конференций TED
### Описание проекта: 
Цель проекта: создание дашборда для анализа проведенных TED-конференций 

Используемые технологии: PostgreSQL, Yandex Datalens

Цель дашборда: проанализировать опыт проведения предыдущих конференций TED для составления плана проведения последующих конференций.

Заказчик: Компания, которая приобрела лицензию на проведение конференций TED.

Ссылка на дашборд: https://datalens.yandex/fbcue77vrabu1
